A simple console-based Python project where the user guesses a randomly generated number between 1 and 50.

---

## Features
- Random number generation
- User input handling
- Loop until correct guess
- Helpful hints (too high / too low)

---

## How to Run
1. Install Python
2. Clone this repository using:
   git clone <your GitHub link>
3. Run the game:
   python game.py
